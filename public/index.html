Reinis Varavs API Server

Available endpoints for document conversion, audio processing, and message retrieval

/n8n/convert-docx

Convert .docx files to Google Docs in your Google Drive. Moves original files to destination folder and creates new Google Docs in source folder. Essential for automation workflows since most platforms cannot extract content from .docx files directly.

Headers:
  • x-api-key: Your API key
  • Content-Type: application/json
Body:
  • from: Source folder ID
  • to: Destination folder ID
  • file_id: Google Drive file ID

/n8n/convert-audio

Convert audio files between various formats. Accepts input formats: flac, m4a, mp3, mp4, mpeg, mpga, oga, ogg, wav, webm. Useful for converting audio to Whisper-1 compatible formats for transcription (mp3, mp4, mpeg, mpga, m4a, wav, webm, ogg, opus).

Headers:
  • x-api-key: Your API key
Body:
  • file: Audio file (multipart/form-data)
Query:
  • format: Output format (optional, defaults to mp3)

/n8n/get-calendar-availability

Fetches available time slots from a specified Google Calendar within a given date range and working hours. Uses a Google Service Account to authenticate and query the calendar. Defaults to Europe/Riga time zone with DST awareness, but can be customized. Useful for booking workflows, appointment schedulers, and automation requiring real-time availability data.

Headers:
  • x-api-key: Your API key
Body:
  • google_calendar_email: Calendar account email
  • google_service_account_key: Service account JSON (object or string)
  • utc: UTC offset in hours (optional, defaults to Europe/Riga current offset)
  • days: Number of days ahead to check (optional, defaults to 7)
  • work_start_hour: Start of workday in 24h format (optional, defaults to 9)
  • work_end_hour: End of workday in 24h format (optional, defaults to 17)
Response:
  • JSON object with available time slots formatted for scheduling workflows
